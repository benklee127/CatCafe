# M1 Manual Test: Core Loop

## Placement
1. Left click empty valid tile -> chair appears snapped.
2. Left click same tile -> no duplicate.
3. Right click occupied tile -> chair removed.
4. Click outside bounds -> no placement.

## AI / Overstim
1. Confirm cats wander over time.
2. Confirm patrons move and get near cats.
3. Confirm cat overstim bars increase on contact.
4. Confirm cat enters retreat at 100.
5. Confirm cat reaches rest area and cooldowns.
6. Confirm cat returns to wander after rest.

## Pass criteria
- No stuck cat states for 3 minutes.
- No script errors in output.
