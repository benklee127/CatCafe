# GAME SPEC

## Pillars
1. Cozy management with readable pressure.
2. Cat welfare first: overstimulation flow control.
3. Rehabilitation and adoption progression.

## Current playable loop
1. Patrons circulate the cafe floor.
2. Cats wander and accumulate overstimulation when patrons are near.
3. At 100 overstimulation cats retreat to rest area.
4. Player places/removes decor on isometric grid to control flow.

## MVP slice scope (locked)
- One room only.
- 6 cat traits.
- 3 patron archetypes.
- 12 decor items.
- Single save slot.

## Out of scope until M3+
- Multiplayer
- Localization
- Extra rooms
- Final art/audio polish
