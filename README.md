# CatCafe

Godot 4.3+ isometric foster cat cafe management prototype.

## Quick start
1. Open project in Godot 4.3+.
2. Run `Scenes/Main.tscn`.
3. Left click to place decor, right click to remove.

## Structure
- `Scenes/` runtime scenes
- `Scripts/` gameplay/data systems
- `Data/` JSON definitions + schemas
- `Tests/` manual scripts
- `Docs/` milestone and scope docs
- `Tools/validate_data.gd` data validator entrypoint
